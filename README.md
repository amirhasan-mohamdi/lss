texplorer


. اجرا:

```bash
texplorer
```

ویژگی‌های جدید:

· ✅ نمایش هم پوشه‌ها و هم فایل‌ها
· ✅ اجرای ابزارها با r + شماره (مثلاً r1)
· ✅ ورود به پوشه با شماره معمولی
· ✅ حالت دستور (Command Mode)
· ✅ تغییر بین نمایش فقط پوشه‌ها یا همه فایل‌ها
· ✅ اجرای خودکار اسکریپت‌های مختلف (sh, py, js, php)
· ✅ نمایش اطلاعات کامل هر فایل

راهنمای استفاده:

· عدد ساده (مثلاً 1): ورود به پوشه
· r + عدد (مثلاً r1): اجرای ابزار
· c: باز کردن ترمینال دستوری
· t: تغییر بین نمایش فایل‌ها
· b: بازگشت به پوشه والد
· h: رفتن به HOME

حالا می‌توانید به راحتی بین پوشه‌ها حرکت کنید و ابزارها را
